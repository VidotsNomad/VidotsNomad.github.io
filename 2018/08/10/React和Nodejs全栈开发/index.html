使用 create-react-app

  • yarn add --save：安装第三方库 redux
  • yarn run eject：弹出配置文件，可以自定义配置 webpack
  • 扩展package.json里的 scripts 字段，扩展yarn run命令

ES6 语法

作用域：

  • ES5 中只有函数作用域和全局作用域(使用 var 定义变量)
  • 定义变量使用 let 代替 var
  • const 定义不可修改的变量
  • let 和 const 定义变量有块作用域({})

字符串

  • 字符串模板：使用反引号，直接写变量${}

函数扩展

  • 参数默认值
  • 箭头函数
  • 展开运算符...

对象扩展

  • Object.keys(obj)：获取对象所有的 key，数组类型
  • Object.values(obj)：获取对象所有的 value，数组类型
  • Object.entries(obj)：返回数组，每个元素的值是[key, value]形式
  • 对象合并：{...obj1, ...obj2, date: "2017"}
  • 结构赋值：let [name, age] = ["matt", 24]、let {name, age} = {name: "matt", age: 24}

类

  • 类是 prototype 的语法糖
  • extends继承
  • constructor构造函数

数据结构

  • Set：元素不可重复
  • Map
  • Symbol

模块化

  • ES6 自带模块化机制
  • import、import {}
  • export、export default
  • Node 目前还不支持，需要用require来加载文件

React 基础知识

Redux 的大概流程

  • 首先通过 reducer 新建 store，随时通过 store.getState 获取状态
  • 需要状态变更，store.dispatch(action)来修改状态
  • Reducer 函数接收 state 和 action，返回新的 state，可以用 store.subscribe 监听每次修改

Redux 默认只处理同步，异步任务需要 react-thunk 中间件，并使用 applyMiddleware 开启 thunk 中间件：

$ yarn add redux-thunk --save

React-redux提供Provider和Connect两个接口来链接，使用方法：

  • Provider 组件在应用最外层，传入 store 即可，只用一次
  • Connect 负责从外部获取组件需要的参数
  • Connect 可以用装饰器的方式来写

React-router：

  • BrowserRouter：包裹整个应用
  • Router：路由对应渲染的组件，可嵌套
  • Link：跳转专用
