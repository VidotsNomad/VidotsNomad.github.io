温习JavaScript的笔记

Posted on 2018-08-02

Chrome 控制台

以表格的形式打印对象：

console.table({a: 1, b: 2});

计算一段代码运行的时间：

console.time('flag-name');
some code here...
console.timeEnd('flag-name');

类型转化

  • 转化为字符串：String()和.toString()
  • 转化为数字：Number()和.parseInt()、.parseFloat()

琐碎知识点

  • ==：值比较；===：值和类型比较；
  • 作用域：var是函数作用域，而不是块作用域

DOM 元素选择

单个节点选择器：

  • getElementById()
  • querySelector()

多个节点选择器：

  • getElementsByClassName()
  • getElementsByTagName()
  • querySelectorAll()

DOM 元素遍历

获取子节点、父节点、兄弟节点

DOM 元素创建

  • 创建元素：document.createElement('p')
  • 添加元素：appendChild()
  • 设置属性：node.setAttribute()

替换、删除 DOM 元素

  • 替换元素：parentNode.replaceChild(newNode, oldNode)
  • 移除元素：remove()和removeChild()

Event 相关

  • Event Bubbling：事件触发时，从子元素到父元素，一层层传递；

  • Event Delegation：

    const delItem = document.querySelector(".delete-item")
delItem.addEventListener('click', e => console.log(e.target));

    //如果 document 中有多个 class 为 delete-item 的元素时，事件监听器只会对第一个元素起作用；为了解决这个问题，就用到 Event Delegation

    const container = document.querySelector('#item-container');
container.addEventListener('click', e => {
if(e.target.className === 'delete-item') {
    console.log(e.target);
}
});

Storage

localStorage 对象(以字符串的形式存储的)：

  • localStorage.setItem(key, value);
  • localStorage.getItem(key);
  • localStorage.removeItem(key);

sessionStorage 与 localStorage 相似
