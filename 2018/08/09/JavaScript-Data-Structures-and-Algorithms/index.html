JavaScript算法与数据结构的笔记

JavaScript 基础部分

  • 对象总是认为是 true：new Boolean(false)放在 if 语句中为 true
  • ==比较不同数据类型的变量：boolean 会转化为数字；字符串与数字比较时，字符串转化为数字
  • 使用 VS Code 来调试 js 程序：首先要开启本地服务器，并且将相关信息配置在.vscode/launch.json文件中
  • JS 函数有一个内置的对象 arguments，类型是数组
  • 定义 get 和 set 函数：在函数名前使用 get 或 set
  • 只有模块导出多个成员时，导入时才需要{}
  • index.d.ts：为 js 库编写 index.d.ts 文件是为了使用 ts 开发时有智能提醒

TypeScript 基础

类型自动推断：

let age: number = 10;等同于let age = 10;

接口(有两种类型概念)：

指明对象必须满足的属性和方法的描述

interface Person {
    name: string,
    age: number
}
function printPerson(person: Person) {}

在面向对象编程中接口

interface Comparable {
    compareTo(b): number;
}

class MyCompare implements Comparable {
    age: number;
    compareTo(b): number {
        return this.age > b.age;
    }
}

泛型：

interface Comparable<T> {
    compareTo(b: T): number;
}

class MyCompare implements Comparable<MyCompare> {
    age: number;
    compareTo(b: MyCompare): number {
        return this.age > b.age;
    }
}
