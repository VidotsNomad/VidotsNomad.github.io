MEAN Stack Front To Back视频学习笔记

Posted on 2018-07-29

观看老外关于 React 和 Node 全栈开发视频的笔记。

VS Code 环境搭建

在设置中配置：

"emmet.includeLanguages": {
    "javascript": "javascriptreact"
},
"emmet.syntaxProfiles": {
    "javascript": "jsx"
},
"terminal.integrated.shell.windows":"xxx\\bash.exe"

安装扩展：

  • ES7 React/Redux/GraphQL/React-Native snippets(直接搜索 ES7)
  • Bracket Pair Colorizer
  • Prettier - Code formatter：右键 - 格式化文件
  • Live Server

基本的 Express 搭建

安装依赖项：

$ yarn add express mongoose passport passport-jwt jsonwebtoken body-parser bcryptjs validator

$ yarn add nodemon --dev

MongoDB 搭建

在MLab上创建数据库：
module.exports = {
    mongoURI: "mongodb://matt:****@ds259241.mlab.com:59241/social-storage"
};

数据库连接：
mongoose.connect(db)
    .then(() => console.log("MongoDB Connected"))
    .catch(err => console.log("err"));
