Electron与React开发桌面应用

Posted on 2018-07-31

Electron 和 React 开发桌面应用的环境搭建

搭建开发环境

安装create-react-app：

$ npm install -g create-react-app

创建 React 应用：

$ create-react-app mook

安装 Electron：

$ yarn add electron --dev

修改package.json文件：

* 在scripts里添加："electron-start": "electron ."
* 添加："main": "public/electron.js"

在public目录下创建electron.js，作为 Electron 的启动文件，在开发模式下，主窗口加载 HTML 文件的方式如下：

mainWindow.loadURL('http://localhost:3000')
//确保React应用已经运行在该地址上

运行程序：

$ yarn run start
$ yarn run electron-start

创建稳定的开发与构建过程

添加electron-builder、concurrently、wait-on：

$ yarn add electron-builder wait-on concurrently --dev

配置开发 scripts：

"electron-dev": "concurrently \"yarn start\" \"wait-on http://localhost:3000 && electron .\""

$ export BROWSER=none //通过设置这个可以不打开浏览器

配置生产 scripts：

"preelectron-pack": "yarn build",
"electron-pack": "build -c.extraMetadata.main=build/main.js"(根据实际产生的文件位置配置)

配置生产的信息(防止打包和 React 的 build 目录冲突)：

"build": {
    "appId": "com.mook",
    "files": [
    "build/**/*",
    "node_modules/**/*"
    ],
    "directories": {
    "buildResources": "assets"
    }
},
"homepage": "./"

修改 main.js(Electron 入口文件)支持开发与生产环境

安装electron-is-dev：

$ yarn add electron-is-dev
